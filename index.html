TreeLogger

TreeLogger is a website used for monitoring surveys of trees growing on a user's properties. With TreeLogger, users can create and manage multiple properties within their account, add and manage trees on those properties, and conduct surveys of their trees. By reviewing past surveys, users can obtain valuable insights into tree growth rates, health patterns, and various other qualities.


This personal project is built entirely using Ruby on Rails with a PostgreSQL database. See the public repo here.

SafetyNet Club

The SafetyNet Club is a platform created by the Harmonic Financial Technology team, designed to offer users convenience and support in their search for insurance products. All members of the SafetyNet Club are provided a complimentary $10,000 Accidental Death and Dismemberment policy, and as long as members maintain their accounts, they are automatically eligible for this complimentary coverage.


In addition to the complimentary policy, the SafetyNet Club offers a marketplace of different types of insurance products from various providers with tools designed to make it easy for users to compare products, helping users find the coverage best suited to their needs.


Augmenting the marketplace, the SafetyNet Club also hosts a social hub where policy holders can connect and discuss the pros and cons of different insurance products, allowing users to make more informed decisions with unbiased insights from other policy holders.

GreenHouse

GreenHouse is a webapp that allows users to create an account and add virtual spaces representing rooms or areas where plants can be nurtured and cared for. Each space is manually rated for light and humidity levels by the user. Users can also add plants to their account, which are assigned light and humidity requirements, in addition to other attributes like "date last watered" and "date last fertilized". Users can drag and drop these plants into their existing spaces, and the app will automatically assess the compatibility of the plants with the space, providing a rating in the form of a red-to-green color scale to indicate how well-suited the plant is for the space. With GreenHouse, users can monitor and maintain their plants' health and growth with ease, while gaining valuable insights into the plants' behavior and requirements.


This fullstack personal project was created using React.js, Ruby on Rails, and PostgreSQL.

See the frontend repo here and the backend repo here.

CLI Jeopardy!

A tribute to the late, great Alex Trebek, CLI Jeopardy! is a single-player game mirroring the offical game of Jeopardy!, played entirely within a computer's command line interface. Each round is generated by fetching questions and answers from a third-party API (jService.io) and randomly hiding double jeopardy bonuses on the board. Players are provided an engaging and immersive Jeopardy! experience fully styled with timings, colors, and graphics. This was my very first program created early in my education as a software developer, showcasing skills in third-party API integration, creating engaging user experiences, and developing stylistic visuals in environments as restrictive as the command line.


Created 100% with native Ruby code, the public repo can be found here.

Expressive Rails Oopsies, "Ero" (Ruby Gem)

"Ero" (Expressive Rails Oopsies) is a Ruby mixin that provides consistent and informative error logging for Rails applications. With this lightweight library, developers can easily include the Ero module in their existing classes or modules and gain access to the .ero method. This method can be called on a class, an instance, or a module and allows for flexible error logging options.


The code and detailed usage instructions can be found in Ero's public repo here.
